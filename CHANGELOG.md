# ztab 1.0.0

## Added

- Animated TAB header and footer system
- Custom player list name formatting
- Built-in placeholders with no PlaceholderAPI required
- Rank prefix and suffix formatting support
- Optional `zSupportExtender` integration
- `/ztab status` command
- `/ztab reload` command

## Features

- Works on Paper `1.19+`
- Lightweight TAB experience for modern servers
- Built-in placeholder support for player, world, ping, TPS, coordinates, and online counts
- Config-based rank styling for cleaner player list presentation

## Notes

- `ztab` is designed to improve TAB list presentation without requiring PlaceholderAPI
- `zSupportExtender` is optional and can be used for wider compatibility workflows if needed
- Built for broad Paper compatibility with a simple setup flow
